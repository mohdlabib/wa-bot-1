*>> BOT MENU <<*
*-menu* -> melihat semua menu

*-limit* -> limit untuk user gratis

*>> FITUR DASAR <<*
*-groupinfo* -> melihat info grup chat
*-everyone* atau *-tagall* -> tag semua orang di grup chat
*-hidetag* -> kirim pesan dan tag semua orang di grup chat

*>> STIKER & MEDIA <<*
*-sticker* atau *-s* -> membuat stiker dari gambar/video/gif
*-smeme* -> membuat stiker dengan teks
*-meme* -> menambah teks ke gambar
*-toimg* atau *-img* -> merubah stiker menjadi gambar
*-stickers* -> melihat stiker random yang tersedia
*-emojis* -> melihat platform emoji ke stiker yang tersedia
*NB :* Untuk stiker random dan emoji ke stiker ada limit harian. Gunakan seperlunya. Donasi ke bot untuk membantu owner mengoptimalkan bot ini.

*>> DOWNLOADER <<*
*-yt* atau *-ytvid* 'url' -> download video dari YouTube
*-ytaudio* 'url' -> download audio dari YouTube
*-tt* atau *-tiktok* 'url' -> download video dan audio dari TikTok
*-ig* 'url' -> download video dari Instagram
*NB :* Untuk semua fitur downloader ada limit harian. Gunakan seperlunya. Donasi ke bot untuk membantu owner mengoptimalkan bot ini.

*>> IBADAH <<*
*-solat* atau *-salat* 'kota' -> lihat jadwal solat
*-doa* all -> lihat semua doa
